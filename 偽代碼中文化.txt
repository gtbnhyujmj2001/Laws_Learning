# 102年律師二試（憲法/行政法）題目 – 偽代碼＋中文註解版

---
## 題目簡述
甲女（大陸地區人民）與乙男（台灣人）結婚後來台，參加合法遊行發言，被內政部以「參與政治活動」為由強制出境、收容，且未給答辯機會。甲女主張其憲法上基本權利受侵害，請分析。

---

## 偽代碼＋中文註解

步驟一：判斷「大陸地區人民」是否有憲法基本權保障
三分法（分為台灣人、外國人、大陸人）爭點。見解分歧，多數認為居留者仍享有部分基本權保障。
def 是否享有憲法基本權(身分):
if 身分 == "大陸地區人民" and 有合法居留:
# 實務上傾向部分保障（人身自由、程序權等）
return True
# 其他身分或非法居留則保障較弱
return False

步驟二：判斷國家措施是否限制基本權
例如強制出境、收容，涉及人身自由、遷徙自由、集會自由、言論自由等
def 限制了哪些基本權(措施):
權利 = []
if "強制出境" in 措施:
權利.append("遷徙自由")
if "收容" in 措施:
權利.append("人身自由")
if "因遊行發言而處罰" in 措施:
權利.append("集會自由")
權利.append("言論自由")
return 權利

步驟三：審查限制是否合法（比例原則、法律保留）
需檢驗有無法律依據、目的正當性、手段必要、損益權衡等
def 基本權限制是否合法(法律依據, 目的, 手段, 是否給予程序保障):
# 是否有明確法律依據？
if not 法律依據:
return "違憲，無法律保留"
# 目的是否正當？
if 目的 != "維護國安、秩序等正當目的":
return "違憲，目的不正當"
# 手段是否必要（有無較輕微侵害的方法）？
if not 手段 == "必要且相當":
return "違憲，手段過當"
# 有無給予程序保障（答辯、訴願等）？
if not 是否給予程序保障:
return "違憲，違反程序正義"
return "合憲"

步驟四：綜合評述（正反意見羅列）
def 綜合分析(身分, 措施, 法律依據, 目的, 手段, 是否給予程序保障):
結果 = []
if 是否享有憲法基本權(身分):
# 肯定說：大陸地區人民居留者，享有基本人權保障
結果.append("甲女可主張憲法基本權")
權利 = 限制了哪些基本權(措施)
# 進一步審查限制是否合法
合憲性 = 基本權限制是否合法(法律依據, 目的, 手段, 是否給予程序保障)
if 合憲性 != "合憲":
結果.append("本案措施違憲：" + 合憲性)
else:
結果.append("本案措施合憲")
else:
# 否定說：非本國人民，僅受有限度保障
結果.append("甲女不具備憲法基本權主體資格")
return 結果

## 你怎麼用這份筆記？

- **考前腦內預演**：  
  直接看偽代碼跑流程，訓練快速拆解題目 → 寫成中文申論。
- **答題時轉換回中文**：  
  根據每個函數前的註解，把邏輯順序、法律語言完整展開。

### 實用小技巧

- 「流程式答題」：每段對應一段論述
- 答案開頭說明「適用法條／爭點」，後面展開要件、理由、程序、結論
- 用偽代碼想清楚→寫答案時不會漏掉步驟
